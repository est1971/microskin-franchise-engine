# Territory Validation Guide

Territories are not certified only because a polygon exists. Validation explicitly checks:

- weighted viable opportunity count
- active clinic rotation feasibility
- 60 to 90 day revisit plausibility
- monthly consultation capacity
- travel efficiency
- cluster confidence
- dead-space penalty
- contractability score

Statuses include:

- `valid`
- `valid but review recommended`
- `weak / merge recommended`
- `oversize / split recommended`
- `invalid`
