# Environment Variables

- `MICROSKIN_APP_NAME`: Application title used in the UI and health payload.
- `MICROSKIN_DATABASE_URL`: Runtime connection string. Sandbox default is SQLite; production should use PostgreSQL with PostGIS.
- `MICROSKIN_EXPORT_DIR`: Output folder for contract packs.
- `MICROSKIN_DATA_CONFIDENCE_THRESHOLD`: Minimum confidence for certification flows.
- `MICROSKIN_CONTRACTABILITY_THRESHOLD`: Review threshold for legally awkward shapes.
